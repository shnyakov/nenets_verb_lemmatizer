ПЕРЕЧЕНЬ ДЕПОНИРУЕМЫХ МАТЕРИАЛОВ
для программы для ЭВМ
«Программа для автоматической лемматизации ненецких глагольных форм»

Обязательные материалы:
1. manage.py
   Точка запуска программы.
2. config/settings.py
   Конфигурация самостоятельного Django-приложения.
3. config/urls.py
   Основная маршрутизация программы.
4. lemmatizer_app/core.py
   Полный исходный текст ядра лемматизации.
5. lemmatizer_app/views.py
   Веб-слой и JSON API программы.
6. lemmatizer_app/forms.py
   Обработка пользовательского ввода.

Дополнительные материалы для однозначной идентификации реализации:
7. lemmatizer_app/urls.py
   Маршрутизация прикладочного модуля.
8. lemmatizer_app/templates/lemmatizer_app/home.html
   Шаблон пользовательского интерфейса.
9. lemmatizer_app/static/lemmatizer_app/app.css
   Стили интерфейса программы.
10. README.md
    Описание назначения, запуска и состава программы.
11. docs/linguistic_basis.md
    Описание лингвистической основы, происхождения правил и границ реализации.
12. legacy/ЛМ_Проект_Шняков.docx
    Раннее текстовое описание исследовательской базы и разработки алгоритма.
13. legacy/Проект_ЛМ_Шняков.ipynb
    Рабочий ноутбук с примерами исследования, предобработки и проверки алгоритма.

Контрольное замечание:
- Исторические материалы в каталоге `legacy/` включаются в архив как справочные и поясняющие материалы по происхождению алгоритма, но основным объектом регистрации остается текущая программа из файлов данного репозитория.
